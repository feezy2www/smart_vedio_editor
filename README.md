# 文本驱动短视频智能高光剪辑系统
端到端可落地的智能剪辑算法项目，输入长视频+文本描述，自动输出符合短视频节奏的高光成片，覆盖视频理解、跨模态匹配、智能剪辑决策全链路。

## 核心特性
- ✅ 工业级视频预处理与抽帧，支持任意格式视频快速读取
- ✅ 基于TransNetV2的镜头边界检测，实现精准的视频镜头级结构化
- ✅ 基于CLIP的文本-视频跨模态匹配，文本驱动高光片段筛选
- ✅ 时序动作定位(TAL)融合，提升高光提取精度
- ✅ 自动剪辑决策与节奏适配，贴合短视频剪辑逻辑
- ✅ 端到端视频渲染，支持自动转场、BGM添加、字幕生成
- ✅ 【差异化亮点】基于单目深度估计的3D镜头运镜分析，优化转场匹配效果

## 技术栈
- 深度学习框架：PyTorch >= 2.0.0
- 核心模型：CLIP、TransNetV2、InternVideo-TAL、MiDaS、Whisper
- 视频处理：Decord、OpenCV、MoviePy、FFmpeg
- 多模态工具：Hugging Face Transformers
